Automatically changes the pdf to a txt file in the terminal.
**IMPORTANT**
IT IS IMPERATIVE TO CHANGE THE DIRECTORY TO THE EXACT FILE OF WHERE THE PDF IS IN THE SHELL FOR THIS TO WORK 
EXAMPLE: PS C:\Users\yukiz\OneDrive - University of Georgia\Desktop\UGA\IEEE\2026_FYC_Projects\Group_2\pdfParser_python>
**INSTALL with python -m pip install pypdf2**
